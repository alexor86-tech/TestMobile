# TestMobile

Минимальное мобильное приложение-счётчик на React Native и Expo.

## Функциональность

- Крупный текст с текущим числом по центру экрана (начальное значение — 0)
- Кнопка "+" — увеличивает значение на +1
- Кнопка "Сброс" — устанавливает значение обратно в 0

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Отсканируйте QR-код в приложении Expo Go на вашем телефоне или запустите на эмуляторе:
   - Для Android: `npm run android`
   - Для iOS: `npm run ios`
   - Для веб-версии: `npm run web`

## Требования

- Node.js (рекомендуется версия 18 или выше)
- npm или yarn
- Expo Go приложение на мобильном устройстве (для тестирования на реальном устройстве)
